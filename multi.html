Test Results of Multiple Variable Linear Regression

We ran a multiple regression to find the relationship between state-wise voting rate and the four metrics defined before that indicates engagement levels. Results can be found in this table. We can see that none of the p-values are less than 0.05, which shows that even when we expand our input to multiple variables using all four measurements, there is no evidence showing that state-wise voting rate can be explained by Twitter user engagement on political matters.
Coefficients P-value
Participation rate 0.011909109 0.807829
Likes -0.000552305 0.229485
Retweets 0.000595507 0.822347
Followers 0.000100375 0.638767
